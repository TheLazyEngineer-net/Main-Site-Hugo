# TheLazyEngineer Hugo Homepage

This repository is the code for thelazyengineer.net.

## Development

The development environment is created by the Nix package manager and the `flake.nix` file.

### Build Environment
```bash
nix develop
```

### Build Site
```bash
serve-site
```
